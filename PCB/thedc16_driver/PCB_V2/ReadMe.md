##PCB记录


###V2
- 使用了新的电路设计，然而并不能保证一定能正常工作
- 并不需要单片机来处理反馈操作了，现在的使用方式和舵机类似
- 尺寸7.5*7.5
- R59 R69为电机对应内阻加上一组MOS管欧姆区内阻的1000倍
- 总算将三个部分完整地分开了
- 电机驱动的H桥部分过孔有点多，可能需要重构
- 新的反电动势公式：测量值=(1k/R59)*实际值*51k/30k+1V偏置，实际值均为正
- 单片机输出的PWM直接与测量值比较，由运放驱动电机

###V1

- 直接使用14年电设的电路，稍做修改，如去掉了会爆炸的钽电容等。
- 依然没有开关
- 尺寸限制在7.5*8cm，两固定孔间距7.5cm，表示由于电机电流较小，元件间干扰不大，所以使用了极大的元件密度
- 接口全部留在中间，如果中间竖起的PCB也自己做的话，可以直接使用6x2的接口
- R59 R69为电机对应内阻的1000倍
